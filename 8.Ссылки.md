## HTML-ссылки

Ссылки позволяют путешествовать с одного сайта на другой или переходить из одного раздела страницы к другому.

Для создания ссылки необходимо воспользоваться парным тегом *\<a\>*. Все, что будет находиться между открывающим тегом *\<a\>* и закрывающим *\</a\>* будет считаться ссылкой, и пользователь сможет кликнуть по этому содержимому. Общий синтаксис создания ссылок следующий:

```html
<a href="URL">текст ссылки</a>
```

Самый важный атрибут элемента *\<a\>* — это атрибут href, что указывает на пункт назначения ссылки.

Текст ссылки — это та часть, которая будет видна читателю.

Нажав на текст ссылки, читатель отправится на указанный URL-адрес.

### Пример

Посмотрим, как можно создать ссылку на Википедию:

```html
<!DOCTYPE html>
<html>
 <body>

  <a href="https://en.wikipedia.org/"> Википедия</a>

 </body>
</html>
```

<a href="https://en.wikipedia.org/"> Википедия</a>

Ссылки будут отображаться следующим образом во всех браузерах:

* Не посещенная ссылка подчеркнута синим цветом.

* Посещенная ссылка подчеркнута и окрашена в фиолетовый цвет.

* Активная ссылка подчеркнута красным цветом.

## Атрибуты тега *\<a\>*

Ниже представлены наиболее важные атрибуты тега *\<a\>*:

* **href** — значением атрибута является URL-адрес документа, на который указывается ссылка. Является обязательным атрибутом, его значение необходимо задать, чтобы ссылка работала;

* **target** — данный атрибут указывает на то, в каком окне должна открываться страница, к которому ведет ссылка. По умолчанию переход происходит в том же окне. Но можно передать в этот атрибут следующие значения:

* **_self** — по умолчанию. Открывает документ в том же окне / вкладке, в котором он был нажат;

* **_blank** — страница открывается в новом окне или вкладке;

* **_parent** — страница загружается во фрейм-родитель;

* **_top** — страница загружается в полное окно браузера.

* **download** — дополняет атрибут href и сообщает браузеру, что ресурс должен быть загружен в момент, когда пользователь нажимает на ссылку, вместо того, чтобы, например, предварительно открыть его (как PDF-файл). Значением атрибута download может быть имя сохраняемого файла. Если значение атрибута downloadне указано, то будет использоваться имя исходного документа.

## Список ссылок

Очень часто на сайтах необходимо создать список, каждый элемент которого должен направить на другую страницу. Например, страница кинопоиска с фильмами:

![](https://ucarecdn.com/c287fcd3-35b0-4796-8a05-cb2a059d5bcb/)

## Абсолютные и относительные ссылки

Существует два типа ссылок:

* абсолютные;

* относительные.

### Абсолютные ссылки

Абсолютная ссылка предполагает указание полного адреса страницы, на которую вы ссылаетесь.

![](https://ucarecdn.com/6eb7bd3b-996c-49bd-8b6e-1c8f09ec4c9d/-/crop/700x226/0,0/-/preview/)

Обычно они начинаются с протокола http:// или https://, далее идет название домена сайта и сама ссылка. Пример абсолютной ссылки:

```html
<a href="https://egoroffartem.pythonanywhere.com/course/python/">
```

Абсолютная ссылка — это адрес ресурса целиком. Обычно такие ссылки ставят на сторонние ресурсы и надеются, что адрес не изменится. Если вы используете абсолютные ссылки на сайте, то при смене структуры папок или перемещении файлов большинство ссылок могут сломаться.

### Относительные ссылки

Абсолютные ссылки, как мы поняли, помогают добавить переходы на сторонние ресурсы. Но часто нам нужно сделать переходы на другие страницы нашего сайта. Такие переходы можно назвать внутренними, и с помощью их вы можете перейти с отображения одного вашего HTML документа на другой. Что для это нужно? Все то же самое — правильно указать адрес. Только теперь мы будет иметь дело с файлами, которые находятся на нашем же компьютере. И здесь самое важное в их месторасположении — где они хранятся относительно вашего текущего документа.

Давайте снова посмотрим на пример со списком фильмов:

```html
<!DOCTYPE html>
<html>
 <body>
  
  <p>Рейтинг фильмов:</p>
  <ul>
      <li><a href="https://www.kinopoisk.ru/film/435/">
        Зеленая миля</a></li>
      <li><a href="https://www.kinopoisk.ru/film/329/">
        Список Шиндлера</a></li>
      <li><a href="https://www.kinopoisk.ru/film/326/">
        Побег из Шоушенка</a></li>
      <li><a href="https://www.kinopoisk.ru/film/448/">
        Форест Гамп</a></li>
  </ul>
 </body>
</html>
```

Каждая такая ссылка введет на сторонний ресурс. Но мы можем создать дополнительные файлы HTML, которые будут описывать отдельный фильм. Создадим в этой же папке два файла: forest_gamp.html и green_mile.html

Относительно нашего файла index.html вновь созданные файлы лежат в той же папке, поэтому мы можем просто указать их названия в атрибуте href.

```html
<!DOCTYPE html>
<html>
 <body>
  
  <p>Рейтинг фильмов:</p>
  <ul>
      <li><a href="green_mile.html">Зеленая миля</a></li>
      <li><a href="forest_gamp.html">Форест Гамп</a></li>
  </ul>
 </body>
</html>
```

<p>Рейтинг фильмов:</p>
  <ul>
      <li><a href="green_mile.html">Зеленая миля</a></li>
      <li><a href="forest_gamp.html">Форест Гамп</a></li>
  </ul>

Если создать папку files на том же уровне, как и файл index.html и поместить туда файлы forest_gamp.html и green_mile.html, то относительные ссылки сломаются. Нужно будет добавить в адрес название папки

```html
<ul>
    <li><a href="files/green_mile.html">Зеленая миля</a></li>
    <li><a href="files/forest_gamp.html">Форест Гамп</a></li>
</ul>
```
